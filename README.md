# clamnumpad

[![](https://jitpack.io/v/nightcrawler-/clamnumpad.svg)](https://jitpack.io/#nightcrawler-/clamnumpad)

Custom number keypad - can be used for PIN entry, currency input etc


### Installation

In your project level build.gradle, add:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
```
In your app level build.gradle:

```
dependencies {
	        implementation 'com.github.nightcrawler-:clamnumpad:Tag'
	}
```
